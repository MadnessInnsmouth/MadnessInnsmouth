Accessibility Statement

Last updated: January 2024

Our Commitment

Demon of Fire is committed to ensuring that this website and all published software are accessible to people with disabilities. Accessibility is not an afterthought — it is a core design requirement for every project.

Standards Conformance

This website aims to conform to the Web Content Accessibility Guidelines (WCAG) 2.1 at Level AA. The following measures have been taken:

  • All pages include a skip-to-main-content link.
  • All interactive elements have descriptive accessible names and appropriate ARIA roles where needed.
  • Keyboard navigation is fully supported across all pages.
  • Focus indicators are clearly visible on all interactive elements.
  • Colour contrast ratios meet or exceed WCAG AA requirements for both normal and large text.
  • All images and icons that convey meaning have alternative text; decorative elements are hidden from assistive technology.
  • Page language is declared so screen readers use correct pronunciation rules.
  • The hamburger navigation menu is fully keyboard and screen-reader accessible.

Software Accessibility

Windows desktop applications published on this site are tested against NVDA, JAWS, and Windows Narrator. All controls expose accurate names, roles, and states via Windows UI Automation (UIA). Each application ships with a detailed keyboard shortcut reference.

Known Limitations

This site is in active development. Some placeholder pages may not yet meet the full WCAG 2.1 AA standard. Issues are being addressed as the site matures. If you encounter a barrier, please report it using the contact method below.

Feedback and Contact

If you experience any accessibility barrier on this website or in any published software, please open an issue on the relevant GitHub repository or contact the developer directly. Accessibility reports are treated as high-priority issues.

Technical Approach

This website is built with plain HTML, CSS, and minimal JavaScript. No third-party tracking scripts, no pop-ups that trap keyboard focus, and no time-limited content that cannot be paused.
