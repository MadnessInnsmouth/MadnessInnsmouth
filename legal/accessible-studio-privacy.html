Accessible Studio – Privacy Policy

Last updated: January 2025

Summary

Accessible Studio performs all video editing, photo editing, and file conversion locally on your device. The only optional network activity occurs when you choose to use the Google Gemini AI assistant feature.

Local Processing

The following data is processed locally and never transmitted:

  • Video, image, and audio files you import into a project
  • Project files and saved timelines
  • App preferences and settings
  • Exported output files

Optional AI Assistant (Google Gemini)

If you choose to enable the AI assistant, you supply your own Google Gemini API key in Settings. When the assistant is active:

  • Natural language commands you type are sent to Google's Gemini API.
  • If you use the photo vision feature, image data may be sent to the Gemini API for analysis.
  • Accessible Studio does not store, log, or transmit your Gemini API key beyond passing it directly to Google's API endpoint.
  • All Gemini API usage is governed by Google's Privacy Policy and your own API account terms.

The AI assistant is entirely optional. The app is fully functional without it using built-in keyword matching.

No Telemetry or Analytics

Accessible Studio contains no analytics SDKs, crash reporters, or usage tracking. No data is sent to MadnessInnsmouth servers at any time.

Children's Privacy

Accessible Studio is not directed at children under 13 and does not knowingly collect data from children.

Changes to This Policy

This policy may be updated from time to time. The "last updated" date above reflects the most recent revision.

Contact

For questions, please open an issue on the Accessible Studio GitHub repository .
