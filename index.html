The Vision

Built for the blind, by a blind developer. This isn't just a media player; it's a statement against "walled garden" limitations. No subscriptions, no tracking, just high-fidelity audio and document management.

Features

High-End Audio

  • 10-band Parametric EQ
  • ±12 Semitone Pitch Shift
  • 0.25x – 4.0x Speed Control
  • Spatial Audio Support

Document Engine

  • PDF, HTML, RTF, Markdown
  • Native Text-to-Speech Ready
  • Night Mode & Custom Fonts
  • Secure-Scoped Bookmarks

Sideloading Guide

Since this app provides features Apple forbids, you won't find it on the App Store. Here is how you install it:

1. Download the IPA

Grab the latest build from the GitHub Releases page.

2. Choose Your Method

  • AltStore (Recommended): Drag the IPA into AltStore. Requires a 7-day refresh.
  • TrollStore: Permanent installation for iOS 14.0 – 16.7 users.
  • Sideloadly: Simple desktop-to-iPhone sideloading.

3. Trust the Developer

Go to Settings > General > VPN & Device Management and trust your Apple ID profile.

Technical Architecture

Built with SwiftUI and AVAudioEngine. The audio chain uses an 8-layer overlap algorithm for the highest quality time-stretching available on iOS.

Player → TimePitch → 10-band EQ → Reverb mix → Output
