MadnessInnsmouth Website Privacy Policy

Last updated: April 2026

This website is a static site with no account system, no ad trackers, and no first-party analytics scripts.

Information collection

The website itself does not ask for personal information and does not set advertising cookies. It does not run third-party analytics scripts.

Hosting

This site is hosted on GitHub Pages. As with most hosting platforms, basic server logs (such as IP address and request metadata) may be processed by the host to deliver content and operate infrastructure.

External links

This site links to external platforms such as GitHub and third-party download tools. Once you leave this site, those services operate under their own privacy policies.

Changes to this policy

If website privacy practices change, this policy will be updated and the "Last updated" date will be revised.

Contact

Questions about website privacy? Open an issue on GitHub (opens in new tab).
