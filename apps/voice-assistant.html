About This Project

Voice Assistant is a desktop application for Windows designed with accessibility as its primary goal. Whether you have limited mobility, a visual impairment, or simply prefer a hands-free workflow, Voice Assistant provides a natural way to interact with your computer.

The application is fully compatible with popular Windows screen readers and announces all actions and responses via your preferred assistive technology.

Features

  • Fully operable via keyboard and screen reader
  • Compatible with NVDA, JAWS, and Windows Narrator
  • Voice-controlled desktop navigation
  • Application launching by voice command
  • Text dictation across any input field
  • Customisable wake word and command set
  • Low-latency local processing — no cloud dependency
  • Detailed audio and visual feedback for every action

System Requirements

  • Windows 10 or Windows 11
  • Microphone (built-in or external)
  • Minimum 4 GB RAM recommended
  • .NET 6 Runtime (bundled with installer)

Installation

Download the installer from the link in the sidebar. Run the setup wizard and follow the on-screen instructions. The installer is fully navigable by keyboard and screen reader.

Accessibility Notes

Every control in Voice Assistant has a descriptive accessible name and role. The application uses Windows UIA (UI Automation) so that screen readers receive live updates as the assistant responds to commands.

If you encounter any accessibility barrier, please open an issue on the project repository.
