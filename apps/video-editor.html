Accessible Video Editor

Windows

A video editor for Windows designed from the ground up to be fully operable with a keyboard and screen reader. Edit, trim, cut, and export video without ever needing a mouse.

About This Project

Accessible Video Editor is the first video editor built with blind and low-vision users in mind as a primary audience. Most video editors are graphically intensive and nearly impossible to use with a screen reader — this project changes that.

Every control is reachable by keyboard, every timeline event is announced by the screen reader, and the UI uses high-contrast modes compatible with Windows accessibility features.

Features

  • 100% keyboard navigable interface
  • Full NVDA, JAWS, and Narrator support via Windows UIA
  • Audio-described timeline — hear clip positions and lengths
  • Keyboard shortcuts for all editing operations
  • Cut, trim, split, merge, and export clips
  • Audio waveform editing with screen-reader announcements
  • High-contrast mode and system theme support
  • Export to common formats (MP4, MKV, WebM)

Keyboard Shortcuts

  • Space — Play / Pause
  • ← / → — Step back / forward one frame
  • Shift + ←/→ — Step back / forward 5 seconds
  • I — Set in-point (start of selection)
  • O — Set out-point (end of selection)
  • Delete — Remove selected clip or region
  • Ctrl + E — Open export dialog
  • Ctrl + Z / Ctrl + Y — Undo / Redo

System Requirements

  • Windows 10 or Windows 11
  • Minimum 4 GB RAM (8 GB recommended for HD editing)
  • .NET 6 Runtime (bundled with installer)
  • FFmpeg (bundled)

Installation

Download the installer from the link in the sidebar. The setup wizard is fully keyboard accessible and announces every step via Windows accessibility APIs.

Accessibility Notes

Accessible Video Editor is tested regularly against NVDA, JAWS, and Windows Narrator. All timeline elements expose their position, duration, and type to assistive technology. The application supports Windows High Contrast Mode.

Accessibility bug reports are a priority. If you find something that doesn't work with your screen reader, please file an issue on the project repository.
