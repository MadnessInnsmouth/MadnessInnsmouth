Accessible Studio

Windows

An NVDA-accessible video editor, photo editor, and file converter. Fully keyboard-navigable, screen reader friendly, and controlled via natural language commands. Optionally powered by Google Gemini for intelligent image descriptions and real-time project control.

Windows Screen-Reader Compatible Keyboard First AI-Powered

About

Accessible Studio is the first all-in-one creative suite built with blind and low-vision users as the primary audience. It combines a timeline-based video editor, a canvas photo editor, and a file converter into one fully accessible app — no mouse required.

Every control is reachable by keyboard, every timeline event is announced by your screen reader, and the interface uses high-contrast modes compatible with Windows accessibility features.

Download

Head to the Releases page on GitHub and download the latest .zip for Windows. Extract it and run Accessible Studio.exe — no install required.

Getting Started

  1. Open the app — you land on the Video Editor tab by default.
  2. Import media — click Import Media (or press Ctrl+I) to add video or image files. Click Import Audio for music and sound effects.
  3. Edit — use the toolbar buttons or type natural language commands in the assistant bar at the bottom (e.g. "trim the first 5 seconds", "make it cinematic").
  4. Save your project — click Save Project and it appears in the Projects panel on the right.
  5. Export — click Export Video (or press Ctrl+E), pick a format and quality, and save your finished video.

Features

Video Editor

  • Timeline editing with trim, split, duplicate, and delete
  • 35+ filter presets (cinematic, vintage, noir, warm, and more)
  • 20+ transition types (fade, dissolve, wipe, slide, zoom…)
  • Text overlays and single/multi-line captions
  • Fade in/out for video and audio independently
  • Clip layering, renaming, and detach/reattach
  • Normalize brightness and volume across all clips
  • Undo/redo, drag-to-move clips, multi-track support

Photo Editor

  • Crop, rotate, flip, and resize
  • Brightness, contrast, saturation, and sharpness sliders
  • Filter presets, background removal, and background blur
  • Remove objects from photos
  • Image overlay and insert

File Converter

  • Video: MP4, AVI, MKV, MOV, WebM
  • Audio: MP3, WAV, FLAC, OGG, AAC
  • Image: PNG, JPG, WebP, BMP, GIF, TIFF
  • Document: DOCX to TXT, DOCX to HTML, TXT to HTML

AI Assistant

  • Natural language control — type commands like "trim the first 5 seconds" or "make it cinematic"
  • Connect Google Gemini (free) in Settings for the full experience: real-time timeline editing, photo vision, and complex multi-step commands
  • Works with built-in keyword matching even without an API key
  • Persistent memory across sessions

Accessibility

  • ARIA labels on every control
  • Live announcements for all timeline events
  • Full keyboard navigation — no mouse required
  • Focus trapping in dialogs
  • High-contrast themes and adjustable text size
  • Tested with NVDA, JAWS, and Windows Narrator
  • "Where Am I?" button for instant location feedback
